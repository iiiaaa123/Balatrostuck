Balatrostuck is a mod for the popular poker roguelike Balatro aiming to add Homestuck-related content to the game - why? Why not!

## Mod additions

- **45 new Jokers** related to various jokes and 
memorable moments from Homestuck!
- **1 new Spectral card** that interacts with the new Aspect system
- **12 Aspect cards** which passively change your run
- **13 Zodiac cards** to level up your played ranks
- **4 Vouchers** - 2 for the Zodiac system, 2 for the Aspect system
- **2 Tags** - 1 for the Zodiac system, 1 for the Aspect system

## Aspect system
The aspect system adds a new subcategory of consumables called Aspects - unlike other consumables, Aspects are have **exclusive** effects, meaning that one Aspect will override another!
